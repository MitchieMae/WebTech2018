Hypertext Markup Language
x Language used to markup documents (i.e. web pages) in the WWW
	o Structure (and content)
	o Presentation (mostly deprecated in favor of style sheets)

x Versions
	o HTML 2.0, 3.2, 4.0
	o HTML 4.01

 Strict – deprecated
 Transitional – still make use the deprecated
 Frameset – using frames
	o HTML 5 (working draft)

x Head
	o title, base, link, meta, style, script

x Body
	o Grouping elements (div, span)
	o Headings (h1 – h6)
	o Paragraphs, line breaks, horizontal rules (p , br , hr)
	o Lists (ul, ol, li, dl, dt, dd, dir, menu)
	o Tables (table, th, tr, td, thead, tfoot, tbody, colgroup, col)
	o Structured text
 		Phrase elements (em, strong, dfn, code, samp, kbd, var, cite, abbr, acronym)
 		Quotations (blockquote, q)
 		Subscripts and superscripts (sub, sup)
 		Preformatted texts (pre)
	o Font styles and alignments (tt, i, b, big, small, strike, s, u, font, basefont, center)
	o Document changes (ins, del)
	o Links and anchors (a)
	o Objects, images, applets (object, img, param, applet)
	o Scripts (script, noscript)
	o Miscellaneous (address, bdo)
	o Frames, noframes, iframe
	o Frameset (for frameset DTD)

Extensible Hypertext Markup Language
x Reformulation of HTML in XML

x Versions
	o XHTML 1.0
	o XHTML 1.1
	o XHTML 1.2
	o XHTML 2.0
	o XHTML 5 (working draft)

x Basic differences between HTML and XHTML
	o All elements have beginning and ending tags
	o All elements are nested properly
	o Elements and attribute names are case sensitive (lowercase)
	o Attribute values are quoted)
	o Attribute values cannot be minimized

x XHTML Doc. Components
	o Document Type Declaration (DOCTYPES)
 HTML 4.01 DOCTYPES
 XHTML 1.0 DOCTYPES
	o Elements
 Tags
 Content (elements)
 Attributes

x id, class, title, alt, lang, intrinsic event attributes (e.g. onClick)
 Character Entity References

x &lt; &gt; &amp; &nbsp; &shy;

x XHTML Elements
	o Block-level and inline (aka text-level) elements
 Block-level elements represent “larger” document structures and may contain inline & other block level elements
 Inline elements contain only data and other inline elements

x Author Styles
	o External style sheets (recommended)
	o Embedded styles
	o Inline styles

x User Style

x User Agent Style (example default style sheet for HTML 4)
